## 快速开始

### 1\. 环境准备

```Bash

# Python 3.10+
pip install -r requirements.txt
```

### 2\. 配置

复制配置示例并填写你的服务器信息，支持配置多区域分布式节点：

```Bash

cp config.example.py config.py
```

核心配置项：

|文件|变量|说明|
|-|-|-|
|`installer/core/activator.py`|`SERVER\_URL`|你的主验证服务器地址，支持多区域分布式节点配置|
|`installer/core/activator.py`|`SECRET\_KEY`|与服务端一致的签名密钥|
|`installer/core/openclaw\_installer.py`|`OPENCLAW\_TGZ\_URL`|OpenClaw 安装包地址，支持多区域镜像源配置|
|`installer/main.py`|`TRAY\_EXE\_URL`|托盘管家下载地址，支持多区域镜像源配置|

### 3\. 开发模式运行

```Bash

python installer/main.py
```

### 4\. 打包成 exe

```Bash

# Windows 上执行
pip install pyinstaller

pyinstaller --onefile --windowed --name "OpenClaw-Installer" \\
  --icon assets/icon.ico \\
  --paths installer \\
  --hidden-import core.activator \\
  --hidden-import core.api\_validator \\
  --hidden-import core.fingerprint \\
  --hidden-import core.node\_installer \\
  --hidden-import core.openclaw\_installer \\
  installer/main.py --noconfirm --clean
```

打包后的 exe 在 `dist/` 目录，可直接分发给对应地区的用户使用。

## 分布式节点部署说明

本安装器原生支持全球分布式部署架构，你可以在不同国家和地区搭建本地镜像节点，为当地用户提供无访问限制的原生OpenClaw安装服务：

1. 在目标地区搭建轻量化服务器，托管OpenClaw安装包、托盘程序与验证服务节点
2. 在安装器配置中，对应修改为该地区的本地镜像地址
3. 打包后的安装器可直接对接本地节点，用户无需任何网络配置，即可实现一键安装
4. 支持节点同步官方仓库更新，确保用户始终使用最新的原生OpenClaw版本

推荐使用 Python FastAPI + SQLite 搭建节点服务，最低配置 1核1GB内存即可稳定运行，适配不同地区的服务器环境。

## 验证服务器

安装器需要配合验证服务器使用，支持分布式节点部署，核心负责：

* 激活码验证（一码一用、过期检查、区域适配）
* 托管多区域OpenClaw安装包镜像，绕过访问限制
* 托管托盘管家exe多区域镜像
* 同步官方OpenClaw更新，确保所有节点版本一致

服务器端代码**不包含在本仓库中**。你需要自己搭建提供以下 API 的服务：

```Plain Text

POST /api/v1/activate    — 验证激活码，支持多区域节点分流
GET  /static/\*.tgz       — 下载 OpenClaw 安装包，支持多区域镜像
GET  /static/\*.exe       — 下载托盘管家，支持多区域镜像
```

## 技术细节

* **GUI**: customtkinter / tkinter
* **打包**: PyInstaller (--onefile)
* **签名验证**: HMAC-SHA256 (payload + timestamp + secret)
* **API 验证策略**: 200/400/403 视为有效，仅 401 视为 Key 无效
* **安装方式**: 从分布式镜像节点下载预打包 tgz，解压到 npm 全局目录
* **Gateway 启动**: PowerShell Start-Process + 完整路径
* **多区域优化**: 自动适配对应地区的镜像源、npm加速源与网络环境
